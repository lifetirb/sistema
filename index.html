List categorias

GET https://www.lifeti.com.br/sistema/restiapi.php?api=categoria

Parameters

ParameterValueDescriptionData Type
sortasc | ascOrder sort attribute ascending or descendingString
orderbyid-categoria |
imagem-categoria |
nome-categoria |
random		Sort collection by object attributeString
id-categoriaLimit result set to items with more specific by `id_categoria`Integer
imagem-categoriaLimit result set to items with more specific by `imagem_categoria`String
nome-categoriaLimit result set to items with more specific by `nome_categoria`String

Example

- https://www.lifeti.com.br/sistema/restiapi.php?api=categoria
- https://www.lifeti.com.br/sistema/restiapi.php?api=categoria&orderby=id_categoria
- https://www.lifeti.com.br/sistema/restiapi.php?api=categoria&orderby=random
- https://www.lifeti.com.br/sistema/restiapi.php?api=categoria&orderby=id_categoria&sort=desc



Retrieve a categoria

GET https://www.lifeti.com.br/sistema/restiapi.php?api=categoria&id-categoria=<id-categoria>

Parameters

ParameterValueDescriptionData Type
id-categoriaUnique identifier for the object and use random for random IDsInteger

Example

- https://www.lifeti.com.br/sistema/restiapi.php?api=categoria&id-categoria=random
